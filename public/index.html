Universal Reconciliation Service

OpenRefine-compatible reconciliation API powered by Google Gemini AI

API Endpoints

GET Service Metadata

GET /api/metadata

Returns service metadata and capabilities.

POST Reconciliation

POST /api/reconcile

Main reconciliation endpoint for entity matching.

GET Entity Suggestions

GET /api/suggest?prefix=alb&type=scientist

Provides type-ahead suggestions for entities.

GET Entity Preview

GET /api/preview?id=entity_123

Returns HTML preview for a matched entity.

POST Data Extension

POST /api/extend

Extends entity data with additional properties.

Setup Instructions

  1. Set your GEMINI_API_KEY in environment variables
  2. Deploy to Netlify or run locally with npm run dev
  3. Configure OpenRefine to use your endpoint URL

Features

  • Full OpenRefine Reconciliation API compliance
  • AI-powered entity matching with Gemini
  • Rate limiting and security headers
  • Comprehensive error handling
  • Multiple data source integration
